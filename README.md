## How to run

Use dockerï¼Œ you can run it easily.

```
docker-compose up
```

open `http://localhost:8000` in your chrome. 

The other way to run it is manual install dependence.

eg: Postgresql, requirements.txt, Redis, Python3 ...

then, run:

```
python3 server.py -port=8000
```
